# My Resume

A Simple Resume builded using web technologies

**Setup**

```
git clone https://github.com/Pabloitto/My-Resume.git
```

**Install dependencies**
```javascript
npm install
```

**Run Dev Mode**
```javascript
npm run dev
```

**Run Prod Mode**
```javascript
npm start
```

**Technologies**

* **Node JS**
* **Express**
* **Angular JS**
* **Puppeteer**
